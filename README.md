### 게시글 필드 정리
- 제목: string
- 장소(optional): string - 추후 지도 상의 위치로 변경될 수 있음
- 약속 시각: datetime
- 종류: enum(밥|커피|술)
- 모집 인원: number
- 모집 상태: enum(모집중|모집완료) - 모집 인원이 차거나 약속 시간이 지나면 '모집 완료'로 자동으로 변경
